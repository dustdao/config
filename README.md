# Config

SushiSwap config monorepo.

## ESLint

```sh
yarn add --dev @sushiswap/eslint-config
```

## Hardhat

```sh
yarn add --dev @sushiswap/hardhat-config
```

## Prettier

```sh
yarn add --dev @sushiswap/prettier-config
```

## Install

```sh
yarn workspaces foreach install
```

## Build

```sh
yarn workspaces foreach build
```
